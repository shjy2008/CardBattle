# Card Battle 🎮

An **strategic card game** built with **Unity (C#)**, where players use cards at hand to control armies and defeat enemies.

---

## Features

- 🎮 Deploy and manage units across three distinct battle zones (Left Flank, Center, Right Flank).  
- ⚔️ Each card has a unique functions (attack, defence, boost, recovor)
- ⌛️ Turn-based gameplay - wisely use the 8 skill points each turn

---

## Tech Stack

- **Engine:** Unity  
- **Language:** C#  
- **Version Control:** Git

---

## How to Run

1. Clone the repository:

        git clone https://github.com/shjy2008/CardBattle.git

2. Open the project in **Unity Hub**

3. Click **Play** in the Unity Editor
